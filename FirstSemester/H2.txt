String
- sentence of characters 
- considered as object and have specific method

String Index
- each character has a specific position for example first character starts at index 0

String Literal 
- a series of characters found enclosed in double quotes and is considered constant

Constructor
- a block of code that utilizes a newly created object

1. Strings() - default constructor to create an empty string
2. String st = new String(String str); - creates new string object to heap area and stores the given value in it
3. String(char chars[]) - uses string object and array of characters in it
4. String(char chars[], int startIndex, int count) - used to create and initialize String object with subrange of a character array
5. String (byte byteArr[]) - used to create a new String object by decoding the given array of bytes
6. String (byte byteArr[], int startIndex, int count) - - used to create and initialize String object with subrange of an array of bytes

String Methods
- used to perform operation on strings

1. charAt() - returns the character present in the specific index
2. trim() - removes the leading and trailling whitespaces from a string
3. replace() - changes the matching occurrences of a text or character in the string with new text or character
4.  equals() - returns true if two strings are identical and false otherwise
5. toLowerCase()/toUpperCase() - changes all characters in the string to lowercase or uppercase
6. length() - returns the number of characters in the string
7. concat() - method returns a new string based on two concatenated or joined strings
8. indexOf() - returns the position in number of the text 
9. compareTo() - compares two strings based on their dictionary order
