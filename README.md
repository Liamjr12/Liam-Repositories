### Real Projects
Practical Java applications applying OOP and Swing concepts:

#### Grading Calculator
A desktop application that calculates a student's grade average across multiple subjects.

| File | Role |
|------|------|
| `Main.java` | Entry point — launches the application |
| `Display.java` | Builds the full GUI: name/section fields, 7 subject combo boxes, grade inputs, and action buttons |
| `UIConstant.java` | Centralizes all UI constants (frame size, fonts, subject list, button dimensions) |
| `Calculation.java` | Handles grade computation logic |

**Features:**
- Input student name and section
- Select up to 7 subjects from a preset list
- Enter corresponding grades
- Compute average grade with a button click
- Revert/reset form
